# Java_Script

A collection of JavaScript practice programs, mini projects, and concept-based examples created while learning JavaScript, DOM manipulation, and frontend development.

## 📌 About This Repository

This repository contains:

* Core JavaScript practice programs
* DOM manipulation examples
* Event handling examples
* Small frontend projects
* Mini games and utility applications

The purpose of this repository is to improve JavaScript fundamentals through hands-on practice.

---

## 📊 Array Practice

1. Clone the repository:

2. Open the project folder.

3. Run any project by opening its `index.html` file in the browser.
